Cubing Tutorials

This website can teach you how to solve a 3x3 and 4x4 cube!

Where should I start?

Start by visting the basic info page to learn more about how the cube works. Then, try learning the beginner method. After mastering this, you can move on to learning CFOP!

What is CFOP?

CFOP is an advanced speedsolving method that most people use to get faster times. It stands for Cross, F2L (First 2 Layers), OLL (Orientation of the Last Layer), and PLL (Permutation of the Last Layer). You can get to know more about this at the CFOP page.
